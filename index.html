cmyth

The following git repositories exist for the cmyth project:

  • cmyth

    cmyth is a portable C language library for communicating with a MythTV server. The library supports most version of MythTV dating back to 2004 and can be used to implement a full-featured MythTV frontend. The repository also incudes a few sample applications, including a FUSE filesystem.

  • mvpmc_iphone

    This is an example MythTV frontend for iOS.

  • lcm_android

    This is an example MythTV frontend for Android.
